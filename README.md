# GW2AccountBrowser
Companion Android app to Guild Wars 2. Players can browse their own bank and storage accounts while in the field.

## Version 0.1
App reads
- Account info
- Wallet
- Bank
This app retains the following data in database
- World
- Guild
- Currency description
- Item
